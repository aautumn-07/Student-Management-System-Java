# 📚 Student Management System (Java)

## 📌 Description

This is a simple console-based Java application designed to manage student records.
It allows users to perform basic CRUD operations like adding, viewing, searching, updating, and deleting student data.

## ✨ Features

* Add a new student (ID, name, marks)
* View all students
* Search student by ID
* Update student marks
* Delete student by ID
* Prevent duplicate student IDs
* Input validation for safe user input
* Menu-driven console interface (runs continuously until exit)

## 🛠 Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)
* ArrayList for dynamic data storage
* Scanner for user input
* Methods for code reusability
* Switch-case statements

## ▶ How to Run

### 1. Open terminal in project folder

Make sure you are inside the folder containing the `.java` file.

### 2. Compile the program
javac StudentManagementSystem.java

### 3. Run the program

java StudentManagementSystem
