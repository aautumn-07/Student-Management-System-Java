# Student Management System (Java)

## Description
This is a simple console-based Java application to manage student records.  
It allows you to add, view, search, and delete student details.

## Features
- Add a new student with ID, name, and marks
- View all students
- Search for a student by ID
- Delete a student by ID
- Simple and easy to use console interface

## Technologies Used
- Java
- OOP concepts
- ArrayList for storing students
